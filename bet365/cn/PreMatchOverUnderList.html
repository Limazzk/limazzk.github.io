PreMatchOverUnderList API For Bet365

# 1) 请求地址

http://api.bet365data.com/bet365/preMatch/overUnder?LangCode=1

# 2) 调用方式：HTTP get

# 3) 接口描述：

  • 接口描述详情

# 4) 请求参数:

# GET参数:

字段名称 字段说明 类型 必填 备注
LangCode 比赛语言 string N（默认为1英语） 详见LangCodeModel

# GET参数:

字段名称 字段说明 类型 必填 备注
Code interface code status int N 详见CodeModel
Data 比赛信息 DataModel N 详见DataModel
Msg interface code message MsgModel N -

# DataModel参数:

字段名称 字段说明 类型 必填 备注
List 比赛列表信息 Array N 详见ListModel
Total 比赛数量总计 int N -
Channel 当前市场名称 string N -
SyncTime 数据同步时间 int64 N -

# ListModel:

字段名称 字段说明 类型 必填 备注
TeamID 比赛队伍唯一ID string N 详见ListModel
TeamIt 可用于下单 string N 比赛数量总计
TeamMr 比赛热度 string N 详见LangCodeModel
TeamName 队伍名称 string N 详见LangCodeModel
Fi 当前比赛唯一ID string N 详见LangCodeModel可用于查询比赛完整详细信息
Pd 详细页面地址 string N 域名+当前信息即可进入官网详细比赛页面
Bc 比赛时间 int64 N 比赛时间跟地区有关系
League 联盟名称 string N 详见LangCodeModel
Over 大Object N 详见OverModel
Under 小 Object N 详见UnderModel
Vi 是否有直播 string N 0为不可用，1为可用

# OverModel参数:

字段名称 字段说明 类型 必填 备注
Types int N 详见ListModel
ID int N 比赛数量总计
Fi string N 详见LangCodeModel
Odd int64 N 详见LangCodeModel

# UnderModel参数:

字段名称 字段说明 类型 必填 备注
Types int N 详见ListModel
ID int N 比赛数量总计
Fi string N 详见LangCodeModel
Odd int64 N 详见LangCodeModel
